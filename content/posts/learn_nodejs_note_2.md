# 运行环境

> 两种,类似 python

- 命令行交互

```bash
$ node
```

```javascript
> console.log("Hello World");
```

- 文件运行(无需编译)

```bash
$ echo 'console.log("Hello World")' > helloworld.js
$ node helloworld.js
```
